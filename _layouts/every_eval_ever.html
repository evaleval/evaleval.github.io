One schema for
every eval ever.

A unified, open metadata standard for AI evaluation results. We're building the infrastructure to store, search, and combine data from leaderboards, research papers, and local runs.

Explore Schema View Repo

Unified Data

Currently, eval data is scattered across PDFs, disparate JSON formats, and proprietary APIs. We parse metadata from different harnesses into a single, queryable schema.

Search & Retrieve

Host the data to allow researchers to search, filter, and combine evaluation results. Compare models across timestamps, architectures, and varying compute levels.

Rigorous Science

Backed by the EvalEval Coalition, we are developing scientifically grounded research outputs and robust deployment infrastructure for broader impact.

The Standard

Data for each model is stored using a standardized JSON Schema. This ensures consistency whether the data comes from Hugging Face, LiveCodeBench, or a local run.

UUID Based

Conflict-free storage for multiple runs.

Timestamped

Track model performance evolution over time.

Flexible Metrics

Supports both continuous scores and level-based ratings.
